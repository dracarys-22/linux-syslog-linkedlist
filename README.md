# Linux Syslog Linked List

Bu proje Linux işletim sistemindeki syslog kayıtlarını okuyarak
linked list veri yapısında saklayan bir uygulamadır.

## Syslog Nedir

Syslog Linux sistemlerinde sistem olaylarının kaydedildiği log mekanizmasıdır.

## Kullanılan Veri Yapısı

Bu projede Singly Linked List kullanılmıştır.

## Programın Çalışması

Program syslog dosyasını okur.
Her log satırı linked list içine eklenir.
Daha sonra liste ekrana yazdırılır.
